DIR

El comando "dir" sirve para ver todas los subdirectorios y archivos que tiene la unidad en la que se ejecuto el codigo
y esta tiene sus variaciones como puede ser:

  1. /a: esta muestra los archivos y los atributos que tiene
  2. /s: muestra los archivos del direcctorio especificadoy todos sus directorios
este comando tiene mas atributos que se pueden usar como complementarios pero no son tan relevantes (por el momento)

CLS

CLS es una etiqueta para limpiar la pantalla de trabajo, la traduccion en ingles seria clean screen

COPY

Esta etiqueta ayuda a copiar un archivo o copiar en lotes en otra ubicacion y tiene atributos que se pueden adicionar a la etiqueta
como por ejemplo:

  1. /b: indica que un archivo es binario
  2. /d: permite que el archivo de destino se cree sin cifrar
  3. /z: copia archivos de red en modo reiniciable
Esta etiqueta se la puede escribir abreviada "cp" tambien se pueden anexar archivos, especificando un unico archivo de destino pero varios archivos de origen
con caracteres comodines o en el siguente formato:
archivo1+archivo2+archivo3 "direccion o ruta de destino"

COPY CON

Esta etiqueta sirve para crear archivos en el formato desedeado como por ejemplo se pueden crear archivos .bat, .txt, .typ y otros
para crear archivos se debe seguir el siguiente fomato:
copy con "el nombre que llevara el archivo".txt
al precionar enter ya se podra redactar lo que se quiera en el archivo con el nombre deseado por el usuario
una ves finalizado el texto, para terminar la escritura se debe precionar ctrl+cy asi ya se habra terminado de redactara el texto.

RENAME

es una etiqueta que sirve para renombrar una carpeta o un archivo, y se debe seguir la siguiente formula para
el correcto funcionamiento: rename "unida:\ubicacion...\ archivo1 archivo2

TYPE

type es una etiqueta que ayuda a ver el contenido de un archivo, con este comando no se puede editar el archivo

MKDIR

esta etiqueta ayuda a crear directorios y subdirectorios en cmd y tambien se la puede escribir como "MD"
tambien su puede crear uno o varios directorios estando en una carpeta en la que no se quiere el archivo usando comillasy seescribede la siguiente manera:
MD(o MKDIR) "Unidad:\directorio raiz..." "nombre1" "nombre2"

RDMIR

Este codigo ayuda a eliminar directorios y sub directorios con o sin archivos dentro de este, pero para eliminar
subdirectorios con archivos se debe usar "/s" usando ese atributo se puede eliminar un subdirectorio con archivos dentro. RD(o RDMIR) /s "directorio a eliminar"

HELP

Esta etiqueta te ayuda a ver las formas en las que se puede usar una etiqueta, desplegando informacion, detalles, atributos que se pueden usar con
la etiqueta que estas buscando. Ahora para ser mas especifico, se puede escribir HELP y la funcion con la que necesites ayuda, de esta forma de dara informacion mas detallada
sobre el codigo que hayas escrito

VOL

esta etuiqueta muestra la etiqueta del volumen del disco y el número de serie, si existen. y se usa de la siguiente manera:
VOL (unidad:)

CD

Esta etiqueta te ayuda a moverte en el directorio que te encuentres subiendo o bajando subdirectorios y se usa de la siguiente manera:

  • CD.. : Te ayuda a bajar de subdirectorio.
  • CD (unidad:subdirectorio): y de esa forma vas a cualquir directorio que desees.

TREE

Esta etiqueta te muestra de manera grafica la estructura de carpetas de unidad o ruta y esta tiene dos atributos
los cuales te ayudan de distinta forma:

  • /f: Este atributo te ayuda a ver los archivos que hay dentro de la unidad que desees.
  • /a: Usa ASCII en lugar de caracteres extendidos.

ATTRIB

esta etiqueta muestra o cambia los atributos de un archivo, este tambien tiene varios atributos que ayudan a especificar tu codigo, aqui los mas relevantes:

  • +: Establece un atributo.
  • -: Borra un atributo.
  • R: Atributo de archivo de solo lectura.
  • H: Atributo de archivo oculto.
  • A: Atributo de archivo de almacenamiento.

DEL

esta etiqueta elimina uno o mas archivos.

  • names: Especifica una lista de uno o más archivos o directorios. Se pueden usar caracteres comodín para eliminar varios archivos.
  • /p: pide confirmacion antes de eliminar un archivo.
  • /f: fuerza la eliminacion de un archivo.
  • /a: seleciona los archivps que se van a eliminar basandose en los atributos
